# STEEL SURFACE INSPECTION SYSTEM USING DEEP LEARNING

An AI-based Computer Vision project for automated steel surface defect inspection using deep learning.

---

## 📌 Project Overview

This project aims to develop a deep learning model capable of detecting whether a steel surface is **Normal** or **Defective**. If a defect is detected, the system identifies the defect type and provides a confidence score.

The project uses the **NEU Surface Defect Dataset** together with normal steel surface images to build an accurate image classification model.

---

## ✨ Key Features

- 🖼️ Steel surface image classification
- ✅ Normal / Defective prediction
- 🔍 Defect type identification
- 📊 Confidence score prediction
- 📈 Model performance evaluation using standard metrics

## 📂 Dataset

- NEU Surface Defect Dataset
- Additional Normal Steel Surface Images

Defect Classes:

- Normal
- Crazing
- Inclusion
- Patches
- Pitted Surface
- Rolled-in Scale
- Scratches
